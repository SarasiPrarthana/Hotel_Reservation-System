Hotel Reservation System
📌 Overview

The Hotel Reservation System is a Java-based project developed in IntelliJ IDEA to automate hotel booking and management. It enables customers to reserve rooms and hotel staff to manage reservations, guest details, and reports efficiently.

🚀 Features

Manage rooms (available, reserved, occupied)
Add, update, and delete customer details
Reservation, cancellation, and modification
Simple billing and payment handling
Generate reports for occupancy and revenue

🛠️ Technologies Used

Language: Java
IDE: IntelliJ IDEA
Version Control: Git & GitHub
